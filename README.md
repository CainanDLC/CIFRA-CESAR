# CIFRA-CESAR
Projeto simples da cifra cesar. após uma aula da solyd one, vi um comentário mostrando um código que fazia cifra césar com uma mensagem digitada pelo usuário. Decidi tentar recriar esse código e entender a lógica por traz dele.
Gostaria de ressaltar que meu código foi totalmente baseado no do criador original, mas me esforcei para entender a lógica e as novas funções por trás.
